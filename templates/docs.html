Quick Start

  1. Add API_ID and API_HASH to Replit Secrets from my.telegram.org
  2. Go to Setup Page and enter your phone number
  3. Enter the verification code from Telegram
  4. Copy the session string and add it to Replit Secrets as SESSION_SECRET
  5. Your bot is now running!

Deploy to Koyeb

For 24/7 uptime, deploy your bot to Koyeb:

  1. Push your code to GitHub
  2. Create a new service on Koyeb
  3. Connect your GitHub repository
  4. Set environment variables: API_ID, API_HASH, SESSION_SECRET, BASE_NAME
  5. Deploy!

For detailed instructions, see the KOYEB_DEPLOYMENT.md file.

Environment Variables

Variable Required Description
API_ID Yes Telegram API ID from my.telegram.org
API_HASH Yes Telegram API Hash from my.telegram.org
SESSION_SECRET Yes Session string generated during setup
BASE_NAME No Your name without status (default: Hashir)
AUTO_REPLY_MESSAGE No Auto-reply message (default: I'll respond when I get online)

Troubleshooting

Bot won't start

  • Check that all environment variables are set correctly
  • Make sure SESSION_SECRET is the full session string
  • Verify API_ID and API_HASH are from my.telegram.org

Name not updating

  • Check the console logs for errors
  • Make sure the bot has permission to update your profile
  • Wait 30-60 seconds for the status check cycle
